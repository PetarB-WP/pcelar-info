Instalirati - Node Sass (Lib Sasss)
GIt Hub:
Node-sass is a library that provides binding for Node.js to LibSass

1) 
Pre instalacije kreiramo package.josn komandom
----> npm init
Nakon ove komande bice kreiran fajl package.json 

2) 
Instaliramo node-sass komandom
npm i node-sass (npm install node-sass)

nakom istalacije izvrsimo izmenu u fajlu package.json
"scripts": {
    "sass": "node-sass -w scss/ -o dist/css/ --recursive"
  },

-w - watch (monitoring foldera scss)
-o - output (update css fajla u folderu dist)

pokrenemo program da kostantno nadgleda promenu u scss-u komandom
npm run sass

INSTALACIJA SASS-a (2)
1) instaliramo Chocolatey package manager
2) choco install sass ( instaliranje sass-a )
3) sass --version ( provera instalacije sass-a )
4) --watch scss/main.scss site/css/main.css ( pokretanje sass-a )
